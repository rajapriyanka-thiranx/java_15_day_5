Learn how to repeat tasks efficiently using for, while, and do-while loops.

🤔 What is a Loop?

A loop is used to execute a block of code repeatedly until a condition becomes false.

Start ➜ Check Condition ➜ Execute Code ➜ Repeat ➜ Stop

🔢 for Loop

The for loop is used when the number of iterations is known. 

for(int i = 1; i <= 5; i++){
    System.out.println(i);
}

Output: 1 2 3 4 5

🔄 while Loop

The while loop executes as long as the condition remains true. 

int i = 1;

while(i <= 5){
    System.out.println(i);
    i++;
}

▶️ do-while Loop

Executes the code at least once before checking the condition. 

int i = 1;

do{
    System.out.println(i);
    i++;
}
while(i <= 5);

📊 Loop Comparison

for Loop

Best when number of repetitions is known.

while Loop

Best when repetitions depend on a condition.

do-while Loop

Runs at least once regardless of condition.

💻 Example: Print Even Numbers

for(int i = 2; i <= 10; i += 2){
    System.out.println(i);
}

💻 Example: Sum of Numbers

int sum = 0;

for(int i = 1; i <= 5; i++){
    sum += i;
}

System.out.println("Sum = " + sum);

Output: Sum = 15

💻 Example: Multiplication Table

int num = 5;

for(int i = 1; i <= 10; i++){
    System.out.println(num + " x " + i +
        " = " + (num * i));
}

⏹️ break Statement

Used to stop the loop immediately. 

for(int i = 1; i <= 10; i++){

    if(i == 5){
        break;
    }

    System.out.println(i);
}

⏭️ continue Statement

Skips the current iteration and moves to the next one. 

for(int i = 1; i <= 5; i++){

    if(i == 3){
        continue;
    }

    System.out.println(i);
}

⚠️ Common Mistakes

❌ Infinite Loop

Forgetting to update the loop variable.

❌ Wrong Condition

Incorrect condition may skip or overrun iterations.

❌ Missing Semicolon

Be careful with loop syntax.
